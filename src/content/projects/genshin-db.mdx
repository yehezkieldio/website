# Overview

genshin-db was early ambitious project of mine during the early days of Genshin Impact and during my tenure as a server administrator for the Discord server of Genshin Impact Indonesia. The goal was to create a simple NPM package that provides JSON data of Genshin Impact characters, weapons, and artifacts.
